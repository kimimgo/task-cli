# Product Requirements Document

## Project: task-cli

---

## Vision
간결하고 빠른 CLI 기반 할일 관리 도구. DevSwarm 병렬 구현 검증용 toy project.

## Philosophy
- **Simple by Design**: 복잡한 기능보다 핵심 기능에 집중
- **TDD First**: 모든 기능은 테스트로 시작
- **Parallel-Friendly**: 독립적 모듈 구조로 병렬 개발 가능

---

## Core Principles
1. **Design 70% / Implementation 30%**: 설계에 충분한 시간 투자
2. **TDD First**: 테스트 주도 개발
3. **Immutable Contracts**: Phase 0 이후 설계 문서 수정 불가

---

## Technical Stack
- **Language**: Python 3.12+
- **Database**: SQLite (로컬 파일)
- **CLI Framework**: argparse (표준 라이브러리)
- **Testing**: pytest

---

## Features
1. `task add "할일 내용"` - 태스크 추가
2. `task list` - 태스크 목록 조회
3. `task done <id>` - 태스크 완료 처리
4. `task delete <id>` - 태스크 삭제

---

## Success Metrics
- 테스트 커버리지 >= 80%
- 린팅 통과 (ruff)
- 타입 체크 통과 (mypy)
